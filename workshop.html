With growth in demand for cloud services, cloud security has emerged as an area of importance. An efficient cloud security system is required for providing reliable cloud services to the users. In general, cloud security deals with policies, technologies, and controls deployed to protect information and infrastructure associated with the cloud.

Issues related to cloud security are faced both by the cloud providers and the cloud consumers. As cloud providers focus on providing a reliable infrastructure, safety of data is often the users' responsibility. It is clear that the security issues have played a major role in hindering acceptance towards cloud computing. Well-known security issues such as data loss, phishing, botnet pose serious threats to organization's data and software. Moreover, the multi-tenancy model and the pooled computing resources in cloud computing have introduced new security challenges that require novel techniques to tackle with. For example, hackers can use cloud to organize botnet as cloud often provides more reliable infrastructure services at a relatively cheaper price for them to start an attack.

The issues to be discussed in the workshop include, but are not limited to:
1. Governance and regulatory compliance gaps
2. Insecure APIs
3. Cloud & CSP migration issues
4. SLA & trust management gaps
5. Logical storage segregation & multi-tenancy security issues
6. Identity management issues
7. Insider attacks
8. Virtualization issues
9. Cryptography and key management
